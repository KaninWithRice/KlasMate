# KlasMate - Academic Repository

KlasMate is a multi-platform academic repository designed to help students organize, share, and manage their academic resources with ease. Connect with your classmates, upload files, and keep your studies organized in one place.

## 🚀 Ways to Access KlasMate

You can access KlasMate on almost any device. Choose your preferred method below:

### 1. Web Version (Best for any device)
The most up-to-date version of KlasMate is always available directly through your browser. No installation required.
*   **Link:** [https://klas-mate.vercel.app](https://klas-mate.vercel.app)

### 2. Desktop Version (Windows .EXE)
For a dedicated desktop experience without the browser clutter.
*   **Installation:** 
    1.  Download and extract the `KlasMate-win32-x64.zip` folder.
    2.  Open the folder and run `KlasMate.exe`.
    3.  *Tip:* Right-click `KlasMate.exe` and select **"Pin to Taskbar"** for quick access.

### 3. Mobile App (Android .APK)
Take your academic files with you on the go.
*   **Installation:**
    1.  Download the `KlasMate.apk` file to your Android device.
    2.  Open the file and select **Install**.
    3.  If prompted, allow installation from "Unknown Sources" in your device settings.

## ✨ Features
*   **Centralized Repository:** Organize your files by subject and semester.
*   **Social Connectivity:** Add friends and share academic resources instantly.
*   **Document Preview:** View PDFs and Office documents (.docx, .pptx) directly within the app.
*   **Secure Auth:** Secure manual sign-up or quick access via Google Login.
*   **Real-time Chat:** Communicate with classmates and share files through integrated messaging.
